Easily create donation/fundraising pages using Windzfare, a powerful WordPress fundraising plugin. Windzfare provides you with an intuitive way to accept donations online through your own WordPress site in woocommerce way.

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 5.6 or greater
* MySQL version 5.0 or greater

= We recommend your host supports: =

* PHP version 7.0 or greater
* MySQL version 5.6 or greater
* WordPress Memory limit of 64 MB or greater (128 MB or higher is preferred)

= Installation =

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==

= Q. With Windzfare, do I still need a theme? =
A. Think of it like this: a theme is like the frame of the picture, and Windzfare is the tool to paint the picture inside the frame.

= Q. Is WP Windzfare free? =
A. Yes it's completely free.
